Finding Opportunities

It is alright to sometimes be without nothing to do. We can take it as a resting period but this cannot last for long as we eventually get bored and look out for something to do to keep busy.

No one wants to be seen as a lazy girl supporting the saying that an idle hand is the devils's workshop. We will always need to find the opportunity to do something no matter how small or big.

Opportunities are always around us. We do not need to look too far to harness them whether it is rendering an helping hand to the little kid next door or helping out with siblings' assignments.

The opportunities are endless.

No matter how you see opportunities, just remember that it is always a way to add something of value to another.

Look out for the opportunities around you today and you will be glad you did.

Creating your First Web page

  • From your text editor or IDE create a new file
  • Save the file as dufuna.html in any directory of your choice
  • Type the following HTML code in the file:
    <!DOCTYPE html>
    <html>
    <head>
    <title>My First Web Page</title>
    </head>
    <body>
    <h1>Welcome to the Dufuna-Fem Code Camp</h1>
    </body>
    </html>
  • Save your changes

There you have it. Your firt web page is ready!
